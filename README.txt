This is a simple and free FileMaker solution to plan and record trips.

Getting started:

Default username and password:
Username: admin
Password: admin

When you log in, you will be presented with the dashboard, with 3 links taking you to each page.

News: Announce new events!
Diary: Record the trip!
Future Planner: Plan the trip!


Q&A:

Q: Is this free?
A: Yes, all you need is an existing FileMaker Pro or Server License.

Q: How do I open the fmp12?
A: You will need FileMaker, which you can download a trial version.
